# Key Features of Amazon EC2

**Amazon EC2 (Elastic Compute Cloud)**, provided by **Amazon Web Services**, offers flexible and scalable cloud computing resources that allow users to run applications efficiently without managing physical servers.

## Core Key Features

---

## 1️⃣ Elastic Scalability

EC2 allows users to scale resources based on workload demand.

✅ Increase instances during high traffic
✅ Decrease instances when demand drops

**Benefit:** Cost optimization and performance efficiency.

---

## 2️⃣ On-Demand Computing

Users can launch virtual servers anytime.

* No hardware purchase required
* Instant server provisioning
* Flexible resource allocation

---

## 3️⃣ Multiple Instance Types

EC2 provides different instance families optimized for workloads:

| Instance Type     | Use Case                    |
| ----------------- | --------------------------- |
| General Purpose   | Balanced workloads          |
| Compute Optimized | High CPU applications       |
| Memory Optimized  | Large databases             |
| Storage Optimized | Data-intensive workloads    |
| GPU Instances     | Machine learning & graphics |

---

## 4️⃣ Secure Cloud Environment

Security is built into EC2 through:

* Security Groups (Firewall)
* Key Pair authentication
* IAM access control
* Network isolation via VPC

---

## 5️⃣ Flexible Storage Options

EC2 supports multiple storage types:

* **Amazon EBS** → Persistent storage
* **Instance Store** → Temporary storage
* **Amazon S3 integration** → Backup & archive

---

## 6️⃣ High Availability

EC2 instances can run across multiple Availability Zones.

✅ Fault tolerance
✅ Disaster recovery
✅ Minimal downtime

---

## 7️⃣ Auto Scaling

Automatically adjusts number of instances depending on traffic.

Example:

* High demand → New instances added
* Low demand → Instances removed

---

## 8️⃣ Load Balancing Support

Elastic Load Balancer distributes traffic across multiple EC2 instances.

**Advantages**

* Prevents server overload
* Improves application performance

---

## 9️⃣ Full Administrative Control

Users get complete control over:

* Operating System
* Installed software
* Network settings
* Security configurations

---

## 🔟 Pay-As-You-Go Pricing

No upfront investment required.

Users pay only for:

* Compute usage
* Storage consumed
* Data transfer

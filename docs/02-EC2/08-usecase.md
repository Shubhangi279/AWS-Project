# Amazon EC2 Use Cases

## Introduction

**Amazon EC2 (Elastic Compute Cloud)** from **Amazon Web Services** provides scalable virtual servers that can be used for multiple real-world workloads such as hosting, development, analytics, AI, and enterprise applications.

EC2 eliminates the need for physical infrastructure while providing full control over computing resources.
